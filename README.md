# Práctica de DFIR - Windows

## Descripción
Esta práctica de Digital Forensics and Incident Response (DFIR) tiene como objetivo analizar una máquina Windows sospechosa de extraer información de la compañía y de presentar comportamientos extraños. Utilizaremos diversas herramientas proporcionadas durante el curso para realizar el análisis y obtener evidencias.

## Parte 1: Análisis de Máquina Windows
1. **Objetivo**: Determinar indicios y evidencias sobre la extracción de información y comportamientos anómalos.
2. **Herramientas Utilizadas**: Lista y descripción de las herramientas usadas (ej. FTK Imager, Autopsy, etc.).
3. **Procedimiento**: Pasos detallados de cómo se ha llevado a cabo el análisis y los resultados obtenidos.

## Parte 2: Práctica de Memoria RAM
1. **Objetivo**: Realizar una adquisición de memoria RAM y ejecutar comandos de análisis.
2. **Herramientas Utilizadas**: Volatility, DumpIt, FTK Imager, etc.
3. **Procedimiento**:
   - Adquisición de Memoria: Pasos seguidos para realizar la adquisición.
   - Comandos de Análisis: Ejecución de al menos dos comandos con Volatility y análisis de los resultados.

## Parte 3: Práctica de Metadatos
1. **Objetivo**: Examinar la modificación de metadatos por diferentes plataformas de mensajería.
2. **Herramientas Utilizadas**: ExifTool, etc.
3. **Procedimiento**:
   - **Inicial**: Extraer y documentar los metadatos de una foto propia.
   - **WhatsApp**: Enviar la foto por WhatsApp y comparar los metadatos.
   - **Telegram**: Enviar la foto por Telegram y comparar los metadatos.
   - **Email**: Enviar la foto por email y comparar los metadatos.
